# Web-encuentro-de-estancieras
Encuentro Estancieras 2027 🚙  SPA interactiva para la gestión del Encuentro Nacional en San Rafael.  Stack: JS (ES6+), HTML5, CSS3.  Features: Registro con LocalStorage, exportación a Excel (SheetJS), consumo de API OpenWeather y fondos dinámicos.  Logic: Dashboard con estadísticas de inscriptos y vehículos mediante reduce.
