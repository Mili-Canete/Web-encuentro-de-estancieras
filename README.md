# 🚙 Encuentro Nacional Estancieras, Jeep y Baqueanos 2027 - San Rafael
Este proyecto es una aplicación web interactiva diseñada para la gestión y promoción del Encuentro Nacional de Vehículos Clásicos en San Rafael, Mendoza. La aplicación permite centralizar la información logística, el registro de participantes y datos en tiempo real para los asistentes.

# 🚀 Funcionalidades Principales
Gestión de Inscripciones: Formulario dinámico que captura datos de participantes y vehículos, utilizando localStorage para la persistencia de datos en el cliente.

Dashboard de Estadísticas: Cálculo en tiempo real del total de personas y vehículos inscriptos mediante métodos de orden superior (reduce).

Integración de API de Clima: Conexión con la API de OpenWeatherMap para proporcionar el estado del tiempo actual en San Rafael, Mendoza.

Exportación de Datos: Funcionalidad para exportar el listado de inscriptos directamente a formato Excel (.xlsx) utilizando la librería SheetJS.

Interfaz Dinámica: Sistema de navegación Single Page Application (SPA) con fondos dinámicos que muestran paisajes icónicos de la región como el Cañón del Atuel y El Nihuil.

# 🛠️ Tecnologías Utilizadas
Frontend: HTML5, CSS3 (Animaciones y Flexbox), JavaScript (ES6+).

Persistencia: LocalStorage API.

Integraciones: OpenWeatherMap API (Fetch API / Async-Await).

Librerías: SheetJS (XLSX) para la exportación de reportes.
